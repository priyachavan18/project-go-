# Ford GoBike Analytics Dashboard

An interactive business intelligence dashboard built using Streamlit for analyzing the Ford GoBike bike-sharing dataset.

## Features

- Executive Dashboard
- Rider Analytics
- Station Analytics
- Time Analysis
- Route Analysis
- Interactive Maps
- Business Insights

## Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- NumPy
- Folium

## Dataset

Ford GoBike Trip Data (2018)
